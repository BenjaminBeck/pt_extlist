options

options

TS-Key:

plugin.tx_ptextlist.settings.listConfig.[yourListId].filters.[yourFilterBoxId].filterConfigs.[yourNumericFilterId].radiobutton.options

Description

Defines explicit options to display in a data filter. If options are set, no data is fetched from the database. There are to variants to define the data. See the example for details.

Datatype: array
CObject: NO
Example 
# There are two variants to define the data

# Variant 1: Simple key / value
options {
	checkedin = Someone Checked in 
	noonecheckedin = Noone Checked In
}

# Variant 2: Use integer keys to define option order and subKeys key / value to define the options
10 {
    key = 1
    value = LLL:EXT:my_ext/locallang.xml:theLabelOfTheOption
}
                
20 {
    ...
}
