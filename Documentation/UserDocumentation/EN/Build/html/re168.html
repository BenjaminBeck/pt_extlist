fieldIdentifier

fieldIdentifier

TS-Key:

plugin.tx_ptextlist.settings.listConfig.[yourListId].filters.[yourFilterBoxId].filterConfigs.[yourNumericFilterId].tagcloud.fieldIdentifier

Description

WARNING: Which one to choose??? Identifier of a defined data field. The filter affects on this data field. If more than on fieldIdentifier is given, the resulting query is linked with or. Either a single identifier of a defined field, a comma separated list of identifiers or '*' to use all defined fields.

Datatype: fieldIdentifier or *
CObject: NO
Example 
// use a single field
fieldIdentifier = cn_short_local

// use a list of fields
fieldIdentifier = cn_short_local, name_local

// use every defined field
fieldIdentifier = *
