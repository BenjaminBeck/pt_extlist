uncachedSessionStorageAdapter

uncachedSessionStorageAdapter

TS-Key:

plugin.tx_ptextlist.settings.listConfig.base.uncachedSessionStorageAdapter

Description

Define the session storage adapter that should be used in uncached mode. Posible values are the classes described below or any other class that implements the interface Tx_PtExtbase_State_Session_Storage_AdapterInterface.

Datatype: String
Posible values: Tx_PtExtbase_State_Session_Storage_SessionAdapter: Default browser session storage adapter. Tx_PtExtbase_State_Session_Storage_FeUserSessionAdapter: TYPO3 fe_user bases session storage.
Default: Tx_PtExtbase_State_Session_Storage_SessionAdapter
CObject: NO
