aggregateRows

aggregateRows

TS-Key:

plugin.tx_ptextlist.settings.listConfig.[yourListId].aggregateRows

Description

Holds the aggregates columns definitions.

Datatype: Associative array
Posible values: All columnIdentifiers
CObject: NO
Example 
aggregateRows {
	10 {
		phoneColumn {
			aggregateDataIdentifier = sumPhone
			renderObj = TEXT
			renderObj.dataWrap (
				 &sum;: <b>{field:sumPhone}</b>
			)
		}
	}
}
