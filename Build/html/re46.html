rendererConfigs

rendererConfigs

TS-Key:

plugin.tx_ptextlist.settings.listConfig.[yourListId].rendererChain.rendererConfigs

Description

A list of chained renderer classes that work on list data structures. The default renderer class uses the column configuration, to render the list of field data in a list of rows and columns. By default this renderer iscalled at position 100. All defined renderer before 100 work on a field data list, while renderer after 100 work on a column list data structure.

Datatype: Array (10,20,30)
CObject: NO
